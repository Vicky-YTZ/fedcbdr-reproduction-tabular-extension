# Milestone 1: FCIL + Simplified GDR Prototype

This repository contains a partial reproduction of the FedCBDR paper.

Currently implemented:
- federated class-incremental learning pipeline
- replay buffer
- feature extraction
- pseudo feature construction
- server-side SVD
- leverage-score-based replay selection

Not yet implemented:
- task-aware temperature scaling (TTS)
- full non-IID Dirichlet partition
- full benchmark reproduction from the paper

Status:
Work in progress.
